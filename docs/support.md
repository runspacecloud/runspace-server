# Support

Use the official RunSpace support page when you need help.

Include:

- A short description of the problem
- The device and client you are using
- The steps that caused the problem
- Any visible error message
- The approximate time the issue happened

Do not include:

- Passwords
- Complete account keys
- Two-factor authentication codes
- Recovery codes
- Private encryption keys

Screenshots are useful, but remove private information before sending them.

For urgent safety concerns, use the available reporting tools and provide
enough information for the moderation team to locate the relevant account,
room or message.
