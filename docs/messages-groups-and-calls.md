# Messages, groups and calls

## Private messages

Private conversations allow you to communicate directly with another
RunSpace user.

Depending on the client version, messages may support:

- Text
- Replies
- Reactions
- Images
- Files
- Audio
- Video

Only open files that you trust.

## Friend requests

A user may need to be added as a friend before all communication features
become available.

You can accept, decline, ignore or block requests using the available
controls.

## Group direct messages

Group DMs allow several users to communicate in one conversation.

Group owners and supported roles may be able to:

- Rename the group
- Add members
- Remove members
- Create invitation links
- Manage voice features

## Voice calls

RunSpace may attempt to establish a direct voice connection between
participants.

Calls can fail when a firewall, mobile network, VPN or restrictive router
prevents the required connection.

Check microphone permission, selected audio devices and network access
when a call does not connect.

## Reporting content

Use the available reporting tools for messages or users that violate the
community rules.

Blocking a user prevents or limits further interaction but does not
automatically submit a report.
